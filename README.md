# 🏦 HooBank - Modern Bank Landing Page

![HooBank Banner](https://raw.githubusercontent.com/adrianhajdin/project_hoobank/main/src/assets/logo.svg)

A next-generation payment method for everything. HooBank is a modern, sleek, and responsive banking landing page built with React, Vite, and Tailwind CSS.

## ✨ Features

- **Modern Design**: High-performance landing page with a sleek, dark-themed aesthetic.
- **Fully Responsive**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **Optimized Performance**: Built with Vite for lightning-fast development and optimized production builds.
- **Reusable Components**: Clean and modular component architecture using React.
- **Tailwind CSS**: Rapid UI development with utility-first styling.

## 🚀 Key Sections

- **Hero Area**: Engaging call-to-action with interactive elements.
- **Stats**: Real-time metrics highlighting user trust and transaction volume.
- **Business Benefits**: Detailed insights into rewards, security, and balance transfers.
- **Billing & Invoicing**: Visual representation of billing management.
- **Card Deal Flow**: Simplified steps to finding better card deals.
- **Testimonials**: Social proof from industry leaders.
- **Client Logos**: Showcasing partnerships with top global companies (Airbnb, Binance, Coinbase, Dropbox).

## 🛠️ Tech Stack

- **React**: Frontend library for building user interfaces.
- **Vite**: Next-generation frontend tooling.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **ESLint**: Pluggable linting utility for JavaScript.

## 📦 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mayarmohamed123/Bank-Modern-App.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Bank-Modern-App
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

### Running the App

- **Development server**
  ```bash
  npm run dev
  ```

- **Build for production**
  ```bash
  npm run build
  ```

- **Preview production build**
  ```bash
  npm run preview
  ```
